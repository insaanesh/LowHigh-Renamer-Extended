# LowHigh Renamer Extended

A Blender addon for managing LOW/HIGH poly pipelines, bake prep, naming, copying and clean scene organization.

## Installation

1. Download the ZIP  
2. Open Blender → Edit → Preferences → Add-ons  
3. Press *Install* and select the ZIP  
4. Enable the addon in the list  
5. Panel appears in **N-panel → LowHigh**
